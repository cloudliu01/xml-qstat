Web based status monitoring of GridEngine clusters, provided by the BioTeam

Download

You can download this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/olesenm/xml-qstat
or 
$ git clone git://github.com/bioteam/xml-qstat

get the source code on GitHub : olesenm/xml-qstat or bioteam/xml-qstat
